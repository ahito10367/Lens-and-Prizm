# Lens-and-Prizm
Терагерцовые призмы и линзы
По чуть эта программа помогает считать распределение терагерцового излучения в воздухе и кремнии. При этом учтено что выходя из кристалла ниобата лития излучения попадает в кремневую призму/линзу а далее воздух. Считается распределение по радиальным углам и интегралл интенсивности.
